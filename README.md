# WebDevFlashcards

WebDevFlashcards is a study-focused web application built to help the founder and other web developers prepare for interviews by reviewing important web development concepts and algorithm patterns for problem-solving.

## Overview

The project is designed as a lightweight learning tool for structured review. It brings together:

- Flashcards for core web development concepts
- Cheatsheets for quick topic reference
- Algorithm pattern pages for problem-solving practice

The goal is to make interview preparation more practical, repeatable, and easier to revisit over time.

## Features

- Review flashcards for front-end and JavaScript concepts
- Study common algorithm and problem-solving patterns
- Browse cheatsheets for quick concept refreshers
- Navigate through a simple, focused learning interface

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Available Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Purpose

This project was created to support consistent interview preparation by giving developers one place to review foundational concepts, reinforce practical knowledge, and revisit common algorithm patterns used in technical interviews.
